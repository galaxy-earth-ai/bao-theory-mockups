# Bao Theory Mockups

A lightweight public gallery for five Bao Theory homepage concepts.

## Live site

- Gallery: `https://galaxy-earth-ai.github.io/bao-theory-mockups/`

## Contents

- `index.html` — shareable landing page / gallery
- `styles.css` — lightweight shared styling for the gallery
- `concepts/` — original standalone HTML mockups
- `assets/` — image asset used for preview/share cards

## Notes

These are concept mockups intended for review and sharing, not a production-ready website.
