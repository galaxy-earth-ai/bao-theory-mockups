Bao Theory / Tailwind rebuild / March 2026

Five remade concepts, same spirit, cleaner execution.

These are new Tailwind CSS implementations of the original Bao Theory homepage directions.

The goal stayed consistent: street but editorial, while keeping menu, location, and schedule effortless to find.
